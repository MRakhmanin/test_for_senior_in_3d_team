## Задание

### Нормализация ориентации головы на КТ головного мозга

**Контекст.** Часть КТ-исследований головного мозга поступает к нам с произвольной ориентацией головы: при укладке пациента в аппарате голова оказывается наклонённой или повёрнутой. В реконструированном 3D-объёме (volume) анатомические оси мозга в итоге не совпадают от исследования к исследованию.

**Проблема.** Наши системы (детекция и сегментация патологий, измерения, сопоставление исследований) рассчитывают на единую, каноническую укладку головы. Произвольная ориентация снижает их точность и воспроизводимость.

**Цель.** Реализовать автоматический препроцессинг-шаг, который приводит голову к единой канонической ориентации **в 3D**: выравнивает объём по всем трём осям вращения так, что любая входная ориентация всегда отображается в одну и ту же стандартную.

**Критерии выравнивания**

- Срединно-сагиттальная плоскость вертикальна и симметрична - нет наклона к плечу.
- Лицо направлено строго вперёд - нет поворота влево/вправо.
- Наклон вперёд/назад отсутствует.

**Вход / выход**

- Вход — КТ-объём в произвольной ориентации.
- Выход — тот же объём, ресэмплированный в каноническую систему координат.

### Структура датасета

```
data
    |___volumes

        |___{fname_n}.nii.gz  # NIFTI формат КТ-скана головного мозга

        |___{fname_1}

            |___{roi_n}.nii.gz  # бинарная маска в NIFTI формате
        |___{fname_n}
```

где `roi_n` — имя класса, предсказываемого в TotalSegmentator.

### Визуализация

Для визуализации данных можно использовать [ITK-SNAP](https://www.itksnap.org/pmwiki/pmwiki.php): в нём удобно смотреть КТ-скан в формате NIFTI и маски к нему.

### Пример чтения на Python

```python
import SimpleITK as sitk
import numpy as np

def read_nifti(path):
    """Читает .nii.gz через SimpleITK.

    Возвращает:
    np.array с шейпом (D, W, H)
    словарь с частью метаданных
    """
    img = sitk.ReadImage(path)
    meta = {
        "spacing":   img.GetSpacing(),     # (x, y, z)
        "origin":    img.GetOrigin(),
        "direction": img.GetDirection(),
        "size":      img.GetSize(),        # (x, y, z)
    }
    return sitk.GetArrayFromImage(img), meta   # array уже в порядке (z, y, x)
```

1. Поворот обязательно осуществляется в 3D-пространстве; решения только для одной 2D-плоскости не принимаются.
